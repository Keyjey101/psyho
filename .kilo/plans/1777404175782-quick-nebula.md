## 3. Инновационные функции

### 3.1 «Эмоциональный дневник» — автоматические записи
> **Суть:** После каждой сессии Ника автоматически генерирует краткую запись в дневник: что обсуждали, какие эмоции были, что удалось осознать. Пользователь видит ленту записей с возможностью дополнить.

- **Ценность:** Визуализация прогресса между сессиями, терапевтический эффект рефлексии
- **Сложность:** 3 дня (backend: LLM-генерация + CRUD, frontend: лента + карточки)
- **Метрика:** +20% возвращаемости D7

### 3.2 «Карта эмоций» — тепловой календарь
> **Суть:** Календарь-heatmap (как GitHub contribution graph), где цвет ячейки = среднее настроение за день. При тапе — раскрывается список сессий и инсайтов этого дня.

- **Ценность:** Наглядная визуализация эмоциональной динамики за месяц/год
- **Сложность:** 2 дня (frontend: SVG heatmap + drill-down)
- **Метрика:** +15% engagement с MoodPage

### 3.3 «Микро-практики дня» — контекстные упражнения от бота
> **Суть:** Каждый день Telegram-бот присылает 1 персонализированное микро-упражнение (30-60 сек), основанное на теме последней сессии. Завершение упражнения = +1 к streak.

- **Ценность:** Ежедневный touchpoint, формирование привычки, bridging между сессиями
- **Сложность:** 3 дня (backend: cron-задача + персонализация, frontend: bot command)
- **Метрика:** +30% DAU

### 3.4 «Письмо из будущего» — e-mail/TG рассылка с инсайтами
> **Суть:** Раз в неделю Ника пишет короткое «письмо» через Telegram-бот: «На этой неделе ты говорил(а) о X, и вот что удалось заметить...». Тон — тёплый, ободряющий, без давления.

- **Ценность:** Soft re-engagement, ощущение непрерывной заботы
- **Сложность:** 2 дня (backend: weekly cron + LLM generation, bot messaging)
- **Метрика:** +15% возврат на 30-day

### 3.5 «Режим присутствия» — живая анимация Ники
> **Суть:** Когда Ника «думает» (стриминг начинается), вместо обычного спиннера показывается анимированный аватар Ники — мягкое «дыхание» или «моргание». Создаёт ощущение живого присутствия.

- **Ценность:** Эмоциональная связь, снижение ощущения «общаюсь с ботом»
- **Сложность:** 2 дня (Lottie/SVG анимация + интеграция в ThinkingIndicator)
- **Метрика:** Качественная — позитивные отзывы пользователей

### 3.6 «Временная капсула» — послания самому себе
> **Суть:** В конце сессии пользователь может записать короткое послание себе будущему. Через N дней (или перед следующей сессией) Ника показывает: «Вот что ты написал(а) себе N дней назад...»

- **Ценность:** Рефлексия, отслеживание изменений, терапевтический инструмент
- **Сложность:** 2 дня (backend: new model + scheduler, frontend: UI записи + отображения)
- **Метрика:** +10% завершённых сессий

### 3.7 «Ситуативные сценарии» — интерактивные разборы
> **Суть:** Библиотека готовых интерактивных сценариев: «Как сказать "нет"», «Подготовка к трудному разговору», «Управление панической атакой». Пользователь проходит сценарий шаг за шагом с Никой.

- **Ценность:** Практическая польза без необходимости формулировать запрос, снижает барьер входа
- **Сложность:** 5 дней (backend: CRUD сценариев + LLM-генерация шагов, frontend: каталог + stepper UI)
- **Метрика:** +25% новых сессий от неактивных пользователей

### 3.8 «Голосовой дневник» — аудио-записи + транскрипция + анализ
> **Суть:** Вместо текста пользователь может записать голосовое сообщение. Ника транскрибирует (Whisper API), анализирует тон и содержание, отвечает с учётом эмоциональной окраски.

- **Ценность:** Снижение барьера (не нужно формулировать текстом), Capture эмоционального тона
- **Сложность:** 4 дня (backend: Whisper integration через ZAI, frontend: audio recorder + waveform)
- **Метрика:** +20% длины сессий (голосовые сообщения длиннее)

### 3.9 «Терапевтический компас» — навигация по подходам
> **Суть:** Интерактивная визуализация: 6 подходов как «острова» на карте. При нажатии — объяснение подхода + 1 демо-упражнение. Ника рекомендует: «Тебе сейчас может помочь [подход] — попробуй?»

- **Ценность:** Образовательный элемент, осознанный выбор подхода, глубокая персонализация
- **Сложность:** 3 дня (frontend: SVG map + анимации, backend: рекомендации)
- **Метрика:** +15% usage ActionPanel

### 3.10 «Система достижений» — бейджи, streaks, milestones
> **Суть:** Gamification: 7-day streak (ежедневные микро-практики), «Первая сессия», «5 сессий», «Выполнил 3 упражнения», «Открыл все подходы», «30-day streak». Бейджи показываются в профиле и SessionEndCard.

- **Ценность:** Мотивация, формирование привычки, социальное доказательство (share)
- **Сложность:** 4 дня (backend: new model + triggers, frontend: badge grid + animations)
- **Метрика:** +25% удержание 7-day

### 3.11 «Общий контекст с психологом» — экспорт для живого терапевта
> **Суть:** Генерация PDF/DOCX «выписки» из работы с Никой: темы, инсайты, упражнения, динамика настроения. Пользователь может показать это своему живому психологу для преемственности.

- **Ценность:** Bridge между ИИ-терапией и живой терапией, доверие профессионалов
- **Сложность:** 3 дня (backend: PDF generation, frontend: кнопка экспорта)
- **Метрика:** Новый сегмент: профессионалы, рекомендующие Нику клиентам

### 3.12 «Теневой чат» — анонимный обмен инсайтами
> **Суть:** Пользователь может опубликовать анонимный инсайт из своей сессии. Другие пользователи видят ленту инсайтов и могут отреагировать (без комментариев — только «Спасибо, тоже про это»). Без идентификации, без связи с аккаунтом.

- **Ценность:** Социальное доказательство («я не один»), virality, чувство community
- **Сложность:** 5 дней (backend: anonymous CRUD + модерация, frontend: лента + reactions)
- **Метрика:** +20% органический рост через шеринг

## 4. Свайп-карусель «Подышать»

### Текущее состояние

`BreathingExercise.tsx` (354 строки) — карусель из 4 дыхательных упражнений. Переключение через кнопки `<` `>`. Анимация через framer-motion `AnimatePresence`. Dot-индикаторы внизу. SVG-визуализация: triangle, square, circle.

### Задача

Добавить **touch/swipe** навигацию для мобильных устройств:

#### Техническая реализация

**Подход:** Использовать `useGesture` из `@use-gesture/react` (уже совместим с framer-motion) или нативные touch events.

**Шаги:**

1. **Установить зависимость** (если `@use-gesture/react` нет в package.json):
   ```
   npm install @use-gesture/react
   ```

2. **Обернуть контейнер карусели** в `animated.div` с gesture handler:
   ```tsx
   import { useDrag } from '@use-gesture/react'

   const bind = useDrag(({ movement: [mx], direction: [dx], velocity: [vx], cancel, active }) => {
     if (!active && (Math.abs(mx) > 50 || vx > 0.5)) {
       if (dx > 0) prev()
       else next()
     }
   }, { axis: 'x', filterTaps: true })
   ```

3. **Добавить визуальный feedback** — при свайпе контент слегка смещается за пальцем:
   ```tsx
   <motion.div
     {...bind()}
     style={{ x: dragX, touchAction: 'pan-y' }}
   >
   ```

4. **Сохранить существующую анимацию** — `AnimatePresence` продолжает работать для кнопок, а swipe добавляется параллельно.

5. **Учесть edge cases:**
   - Не блокировать вертикальный скролл (touchAction: pan-y)
   - Предотвратить swipe во время активного упражнения (опционально)
   - Haptic feedback на смене (navigator.vibrate(10))

6. **Пороги:**
   - Минимальное расстояние свайпа: 50px
   - Минимальная скорость: 0.5
   - Ось: только X

**Файлы для изменения:**
- `frontend/src/components/chat/BreathingExercise.tsx`
- `frontend/package.json` (добавить `@use-gesture/react`)

**Оценка:** 4-6 часов
